

# ПАСПОРТ ПРОЕКТА

| Параметр | Описание |
| :--- | :--- |
| **Название проекта** | **Веб-платформа для мониторинга арбитража ставок финансирования на бирже BingX**<br>*(BingX Funding Arbitrage Monitor)* |
| **Заказчик проекта** | Иванов Денис Юрьевич, \n СПбПУ Петра Великого |
| **Команда проекта** | Шептухин Павел Владиславович |
| **Цели проекта** | 1. **Автоматизация поиска возможностей:** Обеспечить трейдеров инструментом для автоматического отслеживания спредов и ставок финансирования в реальном времени.<br>2. **Снижение рисков:** Предоставить аналитические инструменты для точного расчета доходности и оценки рисков перед входом в сделку.<br>3. **Оперативное информирование:** Реализовать систему мгновенных уведомлений о появлении выгодных рыночных условий. |
| **Задачи проекта** | 1. Реализовать сбор и обновление рыночных данных (ставки, цены Spot/Perp) с биржи BingX в реальном времени (каждые 30 сек).<br>2. Разработать дашборд для визуализации текущих возможностей с расчетом APY и спредов.<br>3. Создать модуль исторических данных с построением интерактивных графиков для анализа стабильности ставок.<br>4. Разработать калькулятор арбитража для симуляции P&L (прибылей/убытков) с учетом комиссий и плеча.<br>5. Реализовать систему настраиваемых алертов (уведомлений) по заданным критериям (Threshold, APY).<br>6. Создать инструменты управления портфелем для трекинга открытых позиций.<br>7. Обеспечить возможность экспорта данных для внешнего анализа. |
| **Видение проекта** | Специализированное веб-приложение, автоматизирующее рутинный процесс мониторинга арбитражных стратегий (Funding Rate Arbitrage) между спотовым и фьючерсным рынками BingX.<br><br>Пользователь получает доступ к единому дашборду, где агрегируются данные по всем торговым парам. Система самостоятельно рассчитывает потенциальную годовую доходность и спреды, избавляя трейдера от ручных вычислений. С помощью встроенного калькулятора пользователь может смоделировать результат сделки до её открытия.<br><br>Для оперативного реагирования предусмотрена система алертов: трейдер задает условия (например, «Ставка > 0.05%»), и платформа уведомляет его при наступлении события. Также доступен функционал ведения портфеля и анализа исторических трендов для оценки надежности стратегии. |
| **Целевая аудитория** | Криптовалютные трейдеры, использующие биржу BingX; арбитражеры, ищущие стратегии с низким уровнем риска (Market Neutral); инвесторы, заинтересованные в получении пассивного дохода от ставок финансирования. |
